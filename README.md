# Node Auth API

Authentication API built with Node.js, Express, and MySQL using a local database for learning backend development.

## About

This project is a backend API created to practice authentication concepts with Node.js, Express, and MySQL.

The goal is to understand how to build a real authentication flow, including user registration, login, password hashing, environment variables, database connection, and protected routes.

## Tech Stack

- Node.js
- Express.js
- MySQL
- JavaScript
- CommonJS

## Features

Planned features:

- User registration
- User login
- Password hashing
- MySQL database connection
- Environment variables
- JWT authentication
- Protected routes
- Basic error handling
- Organized project structure
