# Database Setup

This project uses a local MySQL database called `node_auth_api`.

## Database Structure

The database contains the following tables:

- `users`
- `posts`

### Relationships

- One user can have many posts
- Posts are linked to users through a foreign key (`user_id`)
- Deleting a user will automatically delete all related posts (`ON DELETE CASCADE`)

## Files

### schema.sql

Contains the complete database schema exported from MySQL Workbench.

Includes:
- database creation
- tables
- indexes
- foreign keys
- constraints

### models/userModel.sql

Contains the SQL definition for the `users` table only.

### models/postModel.sql

Contains the SQL definition for the `posts` table only.

### workbench/schema.mwb

MySQL Workbench database model file.

### workbench/schema.mwb.bak

Automatic backup generated by MySQL Workbench.

## How to Import

### Option 1 — Using MySQL Workbench

1. Open MySQL Workbench
2. Open `schema.sql`
3. Execute the script

### Option 2 — Using MySQL CLI

```bash
mysql -u root -p < schema.sql
